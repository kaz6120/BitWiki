method check [line 126]




bool check(

$trackback)





Check for spam using this module.
This method is utilized by a Services_Trackback object to check for spam. Generally this method  may not be overwritten, but it can be, if necessary. This method calls the _checkSource() method  for each source defined in the $_options array (depending on the 'continuose' option), saves the  results and returns the spam status determined by the check.
Tags:







return:Whether the checked object is spam or not.
