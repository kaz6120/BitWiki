KinoWiki-Rev : Revamping KinoWiki
=============================================================

## What is KinoWiki-Rev? ##

KinoWiki is one of the Wiki clones, PHP and SQLite driven wiki engine.
KinoWiki-Rev is built for revamping it.

## System Requirements ##

- OS independent. (Tested on OS X and Ubuntu Linux)
- PHP5.3 or later with SQLite enabled.
